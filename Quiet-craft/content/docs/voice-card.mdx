## Voice Chat Card

A modern, animated card UI designed for real-time voice interactions. Inspired by contemporary collaboration tools, this component is perfect for dashboards, chat apps, and team spaces.

### Features

1. **Dynamic avatars:** Show active participants with smooth transitions.
2. **Interactive controls:** Expand, collapse, and join the chat with a single click.
3. **Responsive design:** Looks great on any device.
